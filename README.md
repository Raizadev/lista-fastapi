Foram desenvolvidas as seguintes funcionalidades:

Criação de novos registos com validação de dados de entrada;

Consulta individual e listagem geral de registos persistidos;

Atualização de dados existentes com verificação de integridade;

Remoção segura de registos, assegurando consistência na base de dados.

Adicionalmente, foram aplicadas boas práticas de organização de código, separação de responsabilidades (camadas de modelo, serviço e controlador) e tratamento adequado de exceções. O objetivo principal desta implementação é garantir uma gestão eficiente, segura e escalável da informação no sistema.
